## 🏃 Getting Started

Want to run the legendary bed locally? Follow the sacred steps:

### 1. Clone the repository

```bash
git clone https://github.com/DevLevelling/chitti-vechhadanam.git
cd chitti-vechhadanam
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the dev server

```bash
npm run dev
```

### 4. Open in browser

Head to [http://localhost:3000](http://localhost:3000) — your screen will now radiate luck.

> **Note:** The page auto-updates as you edit files. Hot reload is blessed too.
